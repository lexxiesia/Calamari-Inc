Patch 5.2 - Patch 5.3: Eden's Verse (i500)

Last Updated: 21st October 2020

Black Mage

Both sets (excluding the SMN/RDM Friendly Set) are exceedingly comparable in estimated output, so pick a set with a speed that you feel most comfortable with/can perform well with.

  • 2.16 GCD: High Speed
    Higher average output, but less variance. Has some volatility in terms of MP ticks in Umbral Ice.
  • 2.33 GCD: High Critical
    Slightly lower average output, but higher variance. Does not run into similar MP tick issues as with the High Speed set.
  • 2.33 GCD: SMN/RDM Friendly Set
    This set allows you to use the SMN/RDM sets without swapping melds.

Red Mage

  • 2.50 GCD: Recommended
    The preferred set for most people.
  • 2.50 GCD: Crit Comp
    Good for compositions with heavy buffs to Critical.

Summoner

Although the sets are arranged in damage potential order (highest to lowest), they are all very viable with a less than 0.4% damage difference between the first and the last.

  • 2.50 GCD: Edenchoir Ring
  • 2.50 GCD: Max Crit Double Tome Ring
  • 2.48 GCD: Augmented Bracelet

Patch 5.0 - Patch 5.1: Eden's Gate (i470) / The Epic of Alexander (Ultimate)

Last Updated: 2nd April 2021

Black Mage

For TEA re-clears, use Ultimate Hvergelmir. If not, just replace it with Edengrace Rod as shown below.

  • 2.26 GCD: Dungeon Set
    Provides comparable output to other sets while also giving extra VIT, which can be very useful in prog. This is the recommended set.
  • 2.20 GCD: High Speed Set
    Slightly more estimated damage, with less VIT than the Dungeon Set.

Red Mage

For TEA re-clears, use Ultimate Vorpal Sword. If not, just replace it with Edengrace Rapier as shown below.

  • 2.50 GCD: Etro
  • 2.48 GCD: Etro

The final step of the i515 Relic in Patch 5.45 allows for adding of substats. The following sets use the i515 Relic and are superior than non-Relic sets in every way.

  • 2.50 GCD: Etro
  • 2.48 GCD: Etro

Summoner

For TEA re-clears, use Ultimate Draconomicon with the third meld slot using a +60 Determination materia. If not, just replace it with Book of Grace as shown below.

  • 2.50 GCD: Etro
  • 2.48 GCD: Etro
